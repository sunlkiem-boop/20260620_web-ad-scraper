# how to test facebook ads post andromeda update

## TL;DR
Mark's exact ad-testing CBO structure post-Andromeda: **3–5 adsets, each holding 2–5 variations of ONE unique concept, $10/ad minimum daily budget with a $50/day floor, fully broad targeting**. The variation/new-concept ratio flipped — it was 80% variations / 20% new concepts pre-Andromeda; now it's 80% new concepts / 20% variations.

## Frameworks & models

- **CBO over ABO.** Set budget at campaign level; let Facebook distribute across adsets. Yes, cannibalization happens — that's intended.
- **One adset = one concept; ads within = variations.** Each adset houses a single new concept; the 2–5 ads inside are variations of that concept. So a 3×3 setup = 3 concepts × 3 variations = 9 ads tested simultaneously.
- **Budget rule:** $10 × (total ads in CBO). 9 ads → $90/day. Hard floor of $50/day regardless of ad count — if you can't afford that, don't run paid ads.
- **Broad-everything targeting:** Advantage+ Placements, age 18–65+, both genders, no interests, no lookalikes, no custom audiences. The adset is intentionally just a *bucket* — the only real difference between adsets is the concept of the ads inside.
- **Sales > other objectives.** Mark almost exclusively runs sales campaigns optimized for purchases. Other objectives "are a complete waste of your time and money" with very rare exceptions.
- **Don't mix images + videos in the same adset.** Video cannibalizes image spend. If you must test both, separate CBO.

## Key principles

- Cannibalization is fine — if Facebook starves a creative, it probably wasn't that good. (Caveat below.)
- Optimize ad-level toggles aggressively to OFF: visual touch-ups, music, text improvements, animation, image generation, Advantage+ creative enhancements, language translation. Multi-advertiser ads = ON. "Optimize Website Destination" = OFF.
- CTA "Learn More" tested slightly better than "Shop Now" — but use Shop Now if going direct-to-cart.
- Launch ads next-day at 6 AM Eastern.
- Attribution window: 7-day click + 1-day view, view-through OFF.
- Mark sometimes tests up to **10 new concepts** for a single product launch. He's testing *more* ads now in total, not fewer — just more of them are net-new ideas.
- The "ratio" framing (80/20) is less useful than thinking about it as **the order**: lead with many new concepts, save variations for after you find a winner.

## Tactics & playbook steps — exact playbook

**Structure**
1. Create new campaign → Sales objective.
2. Campaign-level budget = `max($10 × total ads, $50)` per day.
3. Add 3–5 adsets to the CBO.
4. Each adset = 1 new concept. Inside it, 2–5 variations of that concept.

**Adset settings (all identical across adsets except the ads)**
5. Conversion location: Website.
6. Performance goal: Maximize conversions.
7. Dataset: your active pixel.
8. Conversion event: **Purchase**.
9. Attribution: **7-day click + 1-day view**, view-through OFF.
10. Schedule: next day at **6 AM EST**.
11. Location: US (or your primary market). Remove default extras like Singapore.
12. Age: 18-65+. Gender: All. Languages: broad.
13. Placements: **Advantage+ Placements**. Don't touch anything else.

**Ad-level settings**
14. Format: Single image or video. Multi-advertiser ads = ON.
15. Destination URL: matches any URL in ad copy 1:1. **"Optimize website destination" = OFF.** Display link empty. Browser add-ons none.
16. Disable everything you can: promo codes, branding, site links, products, image generation enhancements, overlays, visual touch-ups, music, text improvements, animation, Advantage+ creative enhancements, language translation.
17. CTA: **Learn More** (default; use Shop Now for direct-to-cart).
18. Set tracking pixels + any UTMs.

**Duplication workflow**
19. Build one ad fully → duplicate twice → swap in variation creatives → you have 3 ads (one concept). Duplicate the adset 2× → swap in concept-specific creatives. Result: 3 adsets × 3 ads = 9 ads at $90/day.

## Counterpoints / caveats

- Sometimes Facebook is wrong. Mark notes occasionally a creative with the lowest cost per conversion gets starved by the algorithm anyway. Not a hard rule.
- The $50/day floor is opinion not law.
- Some advertisers run only bid caps and scale to 6-figure days — both extremes exist.
- "Variations are dead" is overstated — variations are still effective, just not as the primary testing mode. The framing should be ORDER, not RATIO.
- Mark recommends testing images first because they're faster to produce; this isn't dogma.
