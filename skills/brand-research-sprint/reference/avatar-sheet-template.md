# Avatar Sheet Template — Phase 2 Reference

Save as `<save-path>/<Brand>_<Product>_Avatar_Sheets.md` (+ matching `.pdf`).

**Target:** 2–3 avatars. Pick the ones the VOC dossier actually supports — don't force ones the data doesn't back.

**Critical rule:** every quote-field cell is verbatim from the VOC dossier with traceable attribution. If a field's quote isn't in the dossier, do a top-up web search — never invent.

---

## Top-of-file boilerplate

```markdown
# <Brand> <Product> — Customer Avatar Sheets

**Source:** `<Brand>_<Product>_VOC_Dossier.md` (N numbered verbatim quotes + contextual). Every quote-field below is verbatim from the dossier with the source URL traceable through §3 (Pain Points) or the personas in §2.

**Avatar set for <Product>:**

- **PRIMARY:** [Avatar 1 name + 1-line descriptor]
- **SECONDARY:** [Avatar 2 name + 1-line descriptor]
- **SECONDARY:** [Avatar 3 name + 1-line descriptor]

**Avatars NOT targeted for <Product>:**
- [Excluded avatar 1] — [reason]
- [Excluded avatar 2] — [reason]

Where the template's "Monthly Revenue" field reads B2B, I've adapted it to **household income + monthly category-spend** — the consumer equivalent. All quotes are verbatim from the VOC research dossier with source attribution traceable.

**Brand-approved category descriptor used throughout:** *[descriptor]*. **Price:** [$/unit + pack-size math]. **Format:** [container, size].
```

## Per-avatar structure (repeated 2–3 times)

Every avatar gets every block below — use the most recently completed brand's Avatar Sheets as the depth pattern.

### Header

```markdown
# AVATAR N — THE [SHORT NAME] (PRIMARY / SECONDARY)

*[1-paragraph italic descriptor — who they are, what they currently use, what trigger event brought them shopping for the category]*
```

### 🔍 Demographic & General Information

Bulleted list with concrete numbers:

- **Age Range:** [band]
- **Gender:** [% female / % male — segmented by funnel creative or by avatar]
- **Location:** [US-only / US + UK + CA + AU / specific metros / suburban / urban — be specific]
- **Monthly Revenue (household income + category-spend):** [household income range]. Current monthly category spend: **[$/month — THE NUMBER THAT MATTERS]** — they're already spending [comparable to product price], just on products they no longer fully trust.
- **Professional Backgrounds:** [knowledge-worker / blue-collar / mixed — be specific]
- **Typical Identities:** [3–6 verbatim self-descriptions in quotes]

### 🚩 Key Challenges & Pain Points

3 numbered challenges. Each has 3–5 verbatim quotes from VOC §3 as evidence.

```markdown
1. **[Challenge name]**
   - *"verbatim quote"* — Source ([cite link](url))
   - *"verbatim quote"* — Source ([cite link](url))
   - [1–2 sentences contextualizing — what does this mean for the avatar]

2. **[Challenge name]**
   - [quotes + context]

3. **[Challenge name]**
   - [quotes + context]
```

### 🌟 Goals & Aspirations

```markdown
- **Short-Term Goals:**
   - [bullet — what they want in 30–90 days]
   - [bullet]
   - [bullet]

- **Long-Term Aspirations:**
   - [bullet — what they want in 1–10 years]
   - [bullet]
   - [bullet]
```

### 🧠 Emotional Drivers & Psychological Insights

3 bullet-points, each 2–3 sentences. These are the "what kind of copy lands" insights — earned-skepticism, validation vs transformation, identity-driven, mechanism-driven, etc.

### 💬 General Direct Client Quotes

3 verbatim quotes that the avatar would say (or has said in the corpus) in everyday conversation about the category.

### 🚩 Pain Points & Frustrations

3 verbatim VOC-quotes that name a specific competitor pain.

### 🎯 Mindset Quote

3 short mantras / mindset lines — either from the corpus or composite-of-VOC framing. These are the lines that go on a sticker / a kitchen-counter affirmation / a t-shirt.

### 🗣 Quotes on Emotional State and Personal Drivers

3 verbatim quotes about how the customer feels about the journey.

### 📢 Quotes on Emotional Responses to Struggles

3 verbatim quotes about the worst-case experiences they've had.

### 🚀 Quotes on Motivation & Urgency Around Success

3 verbatim quotes about why they're shopping now / what success looks like.

### 🚩 Key Emotional Fears & Deep Frustrations

3 bulleted fears, each 1–2 sentences. These are the conversion gates — the things the brand MUST address before the customer buys.

### 🧠 Emotional & Psychographic Insights

3 bulleted insights, each 2–3 sentences. The "what to do in copy" notes per insight.

### 📌 Typical Emotional Journey

4 stages:
- **Awareness:** [moment the pain becomes nameable]
- **Frustration:** [the failed-alternative cycle]
- **Desperation & Seeking Solutions:** [the Google / Reddit / influencer-comment phase]
- **Relief & Commitment:** [the first sip / first dose / first month — the conversion event]

Each stage is 2–4 sentences with at least one verbatim quote.
